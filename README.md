# Enugu Devfest Website

This website contains all the code and assets for the Enugu Devfest website.

All Data (Speakers, Sponsors, Event Schedule, etc) is being kept in the `/data` directory and arranged in folders according to year of happening.

Images (Most) are being hosted on Cloudinary for better image management and transformations.
